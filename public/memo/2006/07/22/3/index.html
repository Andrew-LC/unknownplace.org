Simple tip for text parsers with Catalyst

for my $parser (qw/Markdown Textile Trac Hatena/) {
    __PACKAGE__->plugin( lc $parser, "Text::$parser" );
}
__PACKAGE__->textile->charset('utf-8');

とかで一括セットアップ。

$c->trac->parse($text) とか使える。
