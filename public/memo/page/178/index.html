なんつっても開発者がにーｔ

非互換あるし。lib/appname/C/ -> lib/appname/Controller/ とか。

C → Controller てのは Helper で生成されるデフォルトのものが変わっただけで、今も昔もどちらの名前でもちゃんと使える。

-short オプションつければ従来どおり。

5.33 → 5.5 は Catalyst 本体内部をいじってるようなプログラム以外のアプリであれば問題なくそのまま動く。
