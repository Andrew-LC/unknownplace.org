いろいろ補足しなくちゃいけないところがあるけど、X-Sendfileをfastcgi経由で使うには、fastcgi設定中で

"allow-x-send-file" => "enable"

を追加する必要あります。

fastcgi.server = (
    "/myapp" => ((
        "socket" => "/tmp/myapp_fastcgi.socket",
        "check-local" => "disable",
        "allow-x-send-file" => "enable",
    ))
)
